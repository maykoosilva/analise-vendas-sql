# Análise de Vendas com SQL

Projeto de análise de dados utilizando PostgreSQL para explorar métricas de vendas e desempenho comercial.

## Objetivo

Analisar o desempenho de vendas e identificar crescimento mensal, performance de vendedores e atingimento de metas.

## Tecnologias

* PostgreSQL
